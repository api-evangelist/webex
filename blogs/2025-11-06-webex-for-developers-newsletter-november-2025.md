<h6>📝 Webex API Recent Changelog Hits</h6><p>Highlighting some important updates recently listed in the Webex API Changelogs:</p>
<p><strong>BREAKING CHANGE IN WEBEX CONTACT CENTER APIs –</strong> October 13, 2025: </p>
<p>New APIs to manage <a>Resource Collections</a> have been added. </p>
<p>Version 3 APIs for managing User Profiles have been added:</p>
<ul>
<li><a>Bulk export User Profiles</a></li>
<li><a>Bulk save User Profiles</a></li>
<li><a>Create a new User Profile</a></li>
<li><a>Delete specific User Profile by ID</a></li>
<li><a>Get specific User Profile by ID</a></li>
<li><a>List user profiles</a></li>
<li><a>Update specific User Profile by ID</a></li>
</ul>
<p>Effective Access control list based on User profile permissions and associated resource collections can now be fetched with this new API - <a>Get specific User Profile ACL by ID</a>.</p>
<p>The following User Profile APIs have been deprecated and will be removed soon - <a>Bulk export User Profiles(Version 1)</a>. Older Version 1 of User Profile APIs have been removed as they are non-functional and no longer relevant: <code>Bulk save User Profiles</code>, <code>Delete specific User Profile by ID</code>, <code>Get specific User Profile by ID</code>, <code>List user profiles</code>, and <code>Update specific User Profile by ID</code>.</p>
<p><a>Full Webex Contact Center API Changelog →</a> </p>
