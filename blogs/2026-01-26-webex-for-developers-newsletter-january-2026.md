<h6>📝 Webex API Recent Changelog Hits</h6><ul>
<li><p><strong>UPDATE January 12, 2026</strong>: Previously, the Meeting Transcripts API returned only Webex Assistant meeting transcripts. This API has now been updated to support transcripts generated by Cisco AI Assistant. If both Webex Assistant and Cisco AI Assistant are enabled for a meeting, the transcript returned by this API will be generated by Cisco AI Assistant. For full details, see <a>Meeting Transcripts</a>.</p>
</li>
<li><p><strong>NEW January 15, 2026</strong>: We’re pleased to announce the release of three new meeting summary APIs, these new endpoints enable developers and admins to programmatically retrieve and manage meeting summaries:</p>
<ul>
<li><a>Get Meeting Summaries</a> API </li>
<li><a>Get Summary for a Compliance Officer</a> API</li>
<li><a>Delete Meeting Summaries</a> API</li>
</ul>
</li>
<li><p><strong>NEW January 19, 2026</strong>: New APIs have been added under <a>User Call Settings</a> to manage a person’s <a>Executive settings</a> to configure a person as: <code>EXECUTIVE</code> or <code>EXECUTIVE_ASSISTANT</code>, then use the new APIs to configure the related settings.</p>
</li>
</ul>
