# Webex (webex)

Cisco Webex is a comprehensive collaboration platform offering APIs for messaging, meetings, calling, devices, and contact center workflows. The Webex Developer Platform enables developers to build integrations, bots, embedded apps, and automations using REST APIs, SDKs, and webhooks. Webex supports OAuth 2.0 authentication and provides separate API surfaces for messaging, video conferencing, cloud calling, admin management, and more.

## Features

| Name | Description |
|------|-------------|
| REST APIs | Comprehensive REST APIs for messaging, meetings, calling, devices, admin, and contact center. |
| OAuth 2.0 Authentication | Secure authentication via OAuth 2.0 authorization code flow. |
| Webhooks | Real-time event notifications via HTTP webhooks for messages, memberships, and meetings. |
| Embedded Apps | Build apps that run directly inside Webex meetings and spaces. |
| Bots | Create chatbots that post content and respond to commands in Webex rooms. |
| Widgets | Embed Webex messaging and calling functionality in external web applications. |
| Guest Issuer | Generate guest tokens to add external users to Webex spaces. |
| Admin API | Full programmatic control over Webex organizations, users, licenses, and policies. |
